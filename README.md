## üîß Usage Instructions

1. **Launch the Application**: After installation, double-click the proxinject-go icon.
2. **Select a Process**: From the list, choose the application you want to route through the proxy.
3. **Configure Proxy Settings**: Input your proxy server details in the settings section.
4. **Inject Proxy**: Click the "Inject" button to start redirecting the selected process through the proxy. You'll see real-time logs as the connection establishes.
5. **Monitor Activity**: Use the log to review connection details and diagnose any issues.
